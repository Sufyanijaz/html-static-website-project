Silent Hill 2 Remake – HTML Static Website Project

Author: Sufyan Ijaz
University: University of Engineering & Technology (UET) Mardan
Project Type: Academic Assignment
Technology Used: HTML only

Project Objective

The objective of this project is to design and develop a complete static website using HTML only, without the use of CSS or JavaScript. The website presents a character-focused encyclopedia of Silent Hill 2 Remake, emphasizing psychological themes, symbolism, and structured content presentation.

Key Features

Fully static website created using pure HTML

Well-structured layout using semantic HTML elements

Internal navigation using anchor links

Use of images, tables, lists, and text formatting

Academic and informational content organization

HTML Elements Used

Structural & Semantic: html, head, body, header, nav, section, article, footer

Text Formatting: h1–h4, p, strong, em, font

Media: img

Navigation: a

Tables & Lists: table, tr, th, td, ul, li

Others: br, hr, div, span

University Declaration

This project was developed as part of an academic requirement at University of Engineering & Technology (UET) Mardan to demonstrate understanding of HTML fundamentals and website structure.
